# NeuralNexus — Website

Starter site built with Next.js + TypeScript + TailwindCSS.

## What’s included
- Next.js app (TypeScript)
- Tailwind CSS + brand tokens
- Logo (SVG) in /public
- Basic pages: Home + components

## Local dev
1. Install
   npm install
2. Dev
   npm run dev
3. Build
   npm run build
4. Start
   npm start

## To push to GitHub (example)
1. Create a repo on GitHub or confirm I create `NeuralNexus-site` under your account.
2. Locally:
   git init
   git add .
   git commit -m "chore: initial NeuralNexus site"
   git branch -M main
   git remote add origin git@github.com:TheeJuniorDev/REPO_NAME.git
   git push -u origin main

## Deploy
I recommend Vercel. Connect the GitHub repo for automatic deploys. Use environment variables later for API keys and member services.
